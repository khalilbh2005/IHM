# Projets IHM — C# / .NET

Série de projets développés en C# / .NET dans le cadre de la matière IHM (Interface Homme-Machine) à l'IUT d'Orsay.

## Projets livrés

- **[TPIHM2 - Quizz]** : Application de quiz interactif avec WPF
- **[TPIHM3 - Arbre]** : Structures de données arborescentes
- **[TPIHM4 - Anagramme]** : Jeu d'anagrammes, gestion d'événements
- **[TPIHM5 - Alarme]** : Système d'alarme avec interface graphique

## Projets en cours


- **[TPIHM6 - Dynamique]** : Composants réutilisables
- **[TPIHM7 - MVVM]** : Implémentation du pattern MVVM
- **MineSweeper** : Jeu de démineur complet

## Stack technique

- C# / .NET
- WPF (Windows Presentation Foundation)
- Pattern MVVM
- Visual Studio
- Git
